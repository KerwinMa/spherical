Spherical.js

Display 360x180 panoramas with HTML5.

Download GitHub

Basic Usage

new Spherical({
    container: 'pano',
    front: 'images/front.jpg',
    back: 'images/back.jpg',
    left: 'images/left.jpg',
    right: 'images/right.jpg',
    top: 'images/top.jpg',
    bottom: 'images/bottom.jpg'
});

For more details, see the API on GitHub.

Notes

Supported in Chrome, Firefox, and iOS.

This was mainly a personal experiment - there are some other existing alternatives you might be interested in - the amazing three.js and the paid but robust krpano.
